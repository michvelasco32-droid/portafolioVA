Ampliar Imagen Interactiva

Aprende a crear un visualizador de imágenes con zoom usando OpenCV

📸 ¿Qué hace este código?

Este programa crea un visualizador interactivo de imágenes que permite hacer zoom in y zoom out en tiempo real. Utilizando la biblioteca OpenCV de Python, puedes ampliar o reducir una imagen mediante comandos de teclado, perfecto para inspeccionar detalles o tener una vista general de la imagen. 

def ampliar():
    import cv2
    # Cargo la imagen en una variable
    img_original = cv2.imread("myhero.jfif")
    # Creando variable para el valor del zoom original
    zoom = 1.0

    while True:
        img = cv2.resize(img_original, None, fx=zoom, fy=zoom)
        cv2.imshow("imagen nueva", img)
        tecla = cv2.waitKey(0)
        
        if tecla == ord('+'):
            zoom = zoom * 1.2
        elif tecla == ord("-"):
            zoom = zoom - 0.2
        elif tecla == ord('o'):
            zoom = 1.0
        elif tecla == ord('x'):
            break

🔍 Explicación Paso a Paso

1
Importar OpenCV
Se importa la biblioteca cv2 (OpenCV) que proporciona todas las herramientas necesarias para el procesamiento de imágenes.
2
Cargar la imagen
La función cv2.imread() carga la imagen "myhero.jfif" y la almacena en la variable img_original. Esta será nuestra imagen base que no se modificará.
3
Inicializar el zoom
La variable zoom comienza en 1.0, lo que representa el tamaño original de la imagen (100%). Este valor se modificará según las acciones del usuario.
4
Bucle infinito
El programa entra en un bucle while True que se ejecutará continuamente hasta que el usuario decida salir presionando la tecla 'x'.
5
Redimensionar la imagen
cv2.resize() crea una nueva versión de la imagen aplicando el factor de zoom actual. Los parámetros fx y fy controlan el factor de escala en los ejes X e Y respectivamente.
6
Mostrar la imagen
cv2.imshow() muestra la imagen redimensionada en una ventana titulada "imagen nueva".
7
Esperar entrada del usuario
cv2.waitKey(0) pausa el programa y espera a que el usuario presione una tecla. El valor 0 indica que esperará indefinidamente.
8
Procesar comandos
El programa evalúa qué tecla fue presionada y ajusta el zoom en consecuencia. La función ord() convierte el carácter a su código ASCII para compararlo.

⌨️ Controles del Programa

+
Aumentar zoom en un 20% (multiplica por 1.2)
-
Reducir zoom en 0.2 unidades
O
Restaurar tamaño original (zoom = 1.0)
X
Salir del programa

💡 Conceptos Clave

cv2.imread()
Lee una imagen desde el disco y la convierte en una matriz NumPy que puede ser manipulada.
cv2.resize()
Cambia el tamaño de una imagen. Puede usar dimensiones específicas o factores de escala (fx, fy).
cv2.imshow()
Muestra una imagen en una ventana. Requiere waitKey() para funcionar correctamente.
cv2.waitKey()
Espera la entrada del teclado. El parámetro indica los milisegundos a esperar (0 = infinito).
